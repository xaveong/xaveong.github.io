Active Project

📅 Daily AI Mailer

GitHub Actions와 Gemini 대형 언어 모델(LLM)을 결합하여 개인 맞춤형 콘텐츠를 생성하고 매일 아침 자동으로 발송하는 스마트 메일링 시스템입니다. 외부 API 연동부터 스케줄링 기반 자동화까지 서버리스(Serverless) 환경에서 안정적으로 구동됩니다.

주요 기능 (Key Features)

📰 맞춤형 테크 뉴스 브리핑

구글 뉴스 RSS 피드에서 지정한 키워드(AI 에이전트 등)의 최신 테크 트렌드를 실시간 수집하고 요약합니다.

🗣️ 체계적인 영어 회화 독학

시작일 기준 날짜를 자동 계산하여 진도에 맞춘 '오늘의 표현 3가지, 예문, 원어민 뉘앙스, 영작 퀴즈'를 제공합니다.

🤖 Gemini 가공 & HTML 서식

Gemini 2.5 모델을 활용해 가독성 높은 폰트, 표, 인용구가 적용된 세련된 HTML 이메일 본문을 동적으로 빌드합니다.

⏰ 무비용 완벽 자동화

별도의 개인 서버 없이 GitHub Actions의 Cron 스케줄러를 통해 매일 정해진 아침 시간에 메일을 발송합니다.

🛠️ 사용 기술 및 도구: Python 3.10 Google Gemini API GitHub Actions Python smtplib HTML/CSS Email Template
